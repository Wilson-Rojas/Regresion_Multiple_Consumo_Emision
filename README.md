# Predicción de Emisiones de CO2 mediante Regresión Múltiple No Lineal

## Descripción del Proyecto
Este proyecto implementa un algoritmo de **regresión múltiple no lineal** para predecir las emisiones de CO₂ de vehículos basándose en sus características técnicas.  
A diferencia de la regresión lineal simple, este enfoque permite capturar **relaciones complejas**, **interacciones entre variables** y **patrones no lineales**, proporcionando un modelo más robusto y realista para el análisis ambiental.

---

## Objetivo
Desarrollar un modelo predictivo avanzado que permita:

- Estimar las emisiones de CO₂ considerando interacciones complejas entre variables.  
- Capturar relaciones no lineales y efectos de umbral.  
- Identificar patrones multivariados que influyen en los niveles de emisión.  
- Proporcionar una herramienta sofisticada para análisis ambiental y diseño de políticas de eficiencia.



## Tecnologías utilizadas
- Python  
- NumPy  
- Pandas  
- Matplotlib / Seaborn  
- Scikit-learn
